Install & Start

Find "Sonorium" in the Add-on Store, click Install, then Start. The addon will appear in your Home Assistant sidebar.

3

Add Your Audio

You have two options for adding audio themes:

Option A: Use the Web Interface
Open Sonorium from your sidebar, go to Themes, and create new themes directly in the app. You can upload audio files via drag-and-drop.

Option B: File System
Create theme folders in /media/sonorium/ with your audio files:

Create a Channel

Open Sonorium from your sidebar, click "New Channel", select a theme and your speakers, then hit Play!

Each channel streams independently, so you can have different soundscapes in different rooms.
